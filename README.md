# Plan your day
***
Have a lot of things to do today?
Create notes for your day to follow, and mark them when done.
The app shows the current weather in your location, covers the background with random nature photo, and 
inspire you with a random citation from famous wise people.
## Motivation
***
* Use **Redux** to control state of the **React app**
* Managing three promise lifecycle actions using **createAsyncThunk**
* Use multiple API in the app
* Use [Netlify](https://www.netlify.com/) serverless functions to store environment variables safely
## Technologies
***
* React
* Redux
* JavaScript (ES6)
* HTML5
* CSS3
## APIs used
* [OpenWeather](https://openweathermap.org/)
* [Unsplash](https://unsplash.com/developers)
* [Quotable](https://github.com/lukePeavey/quotable)
## Setup
* **git clone** {the url to the GitHub repo} or clone your own fork
* **cd** into the new folder
* **npm install**
* **netlify dev** (using instead of npm start to handle serverless functions)
